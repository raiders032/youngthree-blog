## 1. DSR이란?

### 1.1 기본 개념

- **DSR(Debt Service Ratio, 총부채원리금상환비율)**은 차주의 **모든 대출에 대한 연간 원리금 상환액**을 **연간 소득**으로 나누어 계산하는 부채상환능력 지표입니다.
- 차주가 소득 대비 얼마나 많은 대출 상환 부담을 지고 있는지를 **종합적으로 판단**하는 가장 포괄적인 기준입니다.

### 1.2 계산 공식

```
DSR = (모든 대출의 연간 원리금 상환액 ÷ 연간 소득) × 100
```

:::tip[DSR의 핵심]
DSR은 **"소득 대비 총 대출 상환 부담"**을 측정하는 가장 정확한 지표입니다. 모든 대출을 포함하여 차주의 실제 상환능력을 종합 평가합니다.
:::

### 1.3 제외 대상

- 다만 금융당국의 지침에 따라 DSR이 적용되지 않는 대출들도 정해져 있습니다. 
- 전세자금대출이나 실수요자 대상 정책금융상품, 서민 대상 금융상품 등이 대표적입니다.
- 이 대출들은 연간 원리금 상환액이 얼마인지와 상관없이 차주의 DSR에 반영되지 않습니다. 
- 다음 대출들이 DSR의 적용 대상에서 제외되는 대출입니다.
    - 전세자금대출 (전세보증금 담보대출은 제외)
    - 보금자리론 및 특례보금자리론
    - 신생아 특례 구입‧전세자금대출
    - 분양주택에 대한 중도금대출
    - 재건축‧재개발 주택에 대한 이주비 대출, 추가분담금에 대한 중도금 대출
    - 분양 오피스텔에 대한 중도금 대출
    - 서민금융상품 (새희망홀씨, 바꿔드림론, 사잇돌대출, 징검다리론, 대학생‧청년 햇살론 등)
    - 300만 원 이하 소액 신용대출 (유가증권 담보대출 포함)
    - 주택연금 (역모기지론)
    - 정책적 목적에 따라 정부, 공공기관, 지방자치단체 등과 이차보전 등 협약을 체결해 취급하는 대출
    - 자연재해 지역에 대한 지원 등 정부정책 등에 따라 긴급하게 취급하는 대출
    - 보험계약대출
    - 상용차 금융
    - 예적금 담보대출
    - 할부‧리스 및 현금 서비스

## 2. LTV, DTI, DSR 비교

### 2.1 세 지표의 핵심 차이점

| 구분 | LTV | DTI | DSR |
|------|-----|-----|-----|
| **측정 기준** | 담보가치 대비 대출 | 소득 대비 주택대출 상환 | 소득 대비 모든 대출 상환 |
| **계산 범위** | 해당 담보대출만 | 주택담보대출 + 기타대출 이자 | 모든 대출 원리금 |
| **주요 목적** | 담보 안전성 확보 | 주택대출 상환능력 평가 | 종합 상환능력 평가 |
| **도입 시기** | 2005년 (DTI와 함께) | 2005년 8월 | 2018년 하반기 |
| **적용 방식** | 물적 기준 | 소득 기준 (주택대출 특화) | 소득 기준 (종합) |

### 2.2 상호 관계와 적용 순서

```
대출 심사 과정에서의 적용 순서:

1단계: LTV 검토
└─ 담보가치 대비 대출 가능 금액 산정

2단계: DTI 검토  
└─ 소득 대비 주택대출 상환 가능성 확인

3단계: DSR 최종 검토
└─ 모든 대출 포함하여 종합 상환능력 평가
```

:::note[규제의 보완 관계]
세 지표는 서로 다른 관점에서 대출 리스크를 평가하며, **모든 조건을 동시에 만족**해야 대출 승인이 가능합니다.
:::

### 2.3 실제 계산 비교 예시

**차주 정보:**
- 연소득: 8,000만원
- 주택 감정가: 10억원
- 신청 주택담보대출: 7억원
- 기존 신용대출: 5,000만원 (월 상환 50만원)

**각 지표별 계산:**

**1. LTV 계산**
```
LTV = 7억원 ÷ 10억원 × 100 = 70%
→ 일반적인 LTV 한도 내 (대부분 70~80%)
```

**2. DTI 계산**
```
주택담보대출 연간 상환액: 약 5,600만원 (7억원, 금리 5%, 30년)
기타대출 연간 이자: 400만원 (5,000만원 × 8%)
DTI = (5,600만원 + 400만원) ÷ 8,000만원 × 100 = 75%
→ DTI 한도 초과 가능성 (일반적으로 60~70%)
```

**3. DSR 계산**
```
모든 대출 연간 상환액: 5,600만원 + 600만원 = 6,200만원
DSR = 6,200만원 ÷ 8,000만원 × 100 = 77.5%
→ DSR 한도 대폭 초과 (일반적으로 40~60%)
```

## 3. 현행 DSR 규제 현황

### 3.1 금융기관별 DSR 한도

| 금융기관 유형 | DSR 한도 | 적용 대상 |
|--------------|----------|----------|
| **1금융권 (시중은행)** | 40% | 모든 대출 |
| **2금융권 (저축은행 등)** | 60% | 모든 대출 |
| **3금융권 (상호금융)** | 70% | 지역별 차등 |
| **비은행권 일부** | 적용 제외 | 고금리 대출 |

### 3.2 지역별 차등 적용

**수도권 (서울, 경기, 인천):**
- 더 엄격한 DSR 적용
- 투기지역/투기과열지구: 추가 강화

**비수도권:**
- 상대적으로 완화된 기준
- 지역 경제 여건 고려

### 3.3 예외 상황

- **생애최초 주택구입자**: DSR 우대 (5~10%포인트 완화)
- **다자녀 가구**: 추가 우대 혜택
- **무주택자**: 일반적으로 우대 적용

## 4. DSR 관리 전략

### 4.1 DSR 개선 방법

**1. 소득 증대**
- 부업이나 사업소득 추가 확보
- 배우자 소득 합산 (연대보증 등)
- 상여금, 성과급 등 추가 소득 증빙

**2. 기존 대출 정리**
- 고금리 신용대출 우선 상환
- 대출 통합(대환대출) 활용
- 만기 연장으로 월 상환액 감소

**3. 대출 구조 조정**
- 원리금균등상환 → 원금균등상환
- 거치기간 활용으로 초기 부담 경감
- 중도상환 계획 수립

### 4.2 DSR 한도 내 최대 대출 받기

**예시: 연소득 1억원, DSR 40% 기준**
```
최대 연간 상환 가능액: 1억원 × 40% = 4,000만원
최대 월 상환 가능액: 약 333만원

가능한 대출 규모 (금리 4%, 30년 기준):
→ 약 6억 2천만원
```

### 4.3 실무 활용 팁

**DSR 계산 시 주의사항:**
- **소득 산정**: 세전소득 기준, 안정적 소득만 인정
- **상환액 산정**: 원리금 합산, 변동금리는 스트레스 테스트 적용
- **기존 대출**: 모든 금융기관 대출 포함

**대출 신청 전 체크리스트:**
1. ✅ 현재 DSR 계산해보기
2. ✅ 목표 DSR과 차이 확인
3. ✅ 필요시 기존 대출 정리
4. ✅ 소득 증빙 자료 준비

:::tip[DSR 최적화 전략]
**DSR을 효과적으로 관리하려면 "소득 늘리기"와 "대출 줄이기"를 동시에 진행하는 것이 가장 효과적입니다.**
:::
