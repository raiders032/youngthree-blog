## 1. DTI란?

### 1.1 기본 개념

- **DTI(Debt To Income ratio, 총부채상환비율)**는 **주택담보대출에 특화된** 부채상환능력 지표입니다.
- 차주의 **주택담보대출 원리금 상환능력**을 소득 기준으로 평가하여 적정 대출 한도를 설정하기 위해 도입되었습니다.

### 1.2 계산 공식

```
DTI = (해당 주택담보대출의 연간 원리금 상환액 + 기타부채의 연간 이자상환액) / 연소득 × 100

포함 항목:
- 주택담보대출: 원리금 상환액 (원금 + 이자)
- 기타 대출: 이자 상환액만 (원금 제외)
```

:::tip[DTI의 특징]
DTI는 **주택담보대출에 집중**한 지표로, 해당 주택대출의 원리금은 모두 포함하지만 다른 대출은 이자만 포함하는 것이 특징입니다.
:::

## 2. 도입 배경과 목적

### 2.1 도입 경과

- **2005년 8월**: LTV 규제 강화의 후속조치로 도입
- **배경**: LTV 규제만으로는 차주의 실제 상환능력을 제대로 평가하기 어려웠음

### 2.2 LTV 규제의 한계와 DTI 도입 필요성

**LTV 규제의 문제점:**
- 차주의 **소득 수준과 무관**하게 주택가격에만 비례하여 대출 한도 결정
- 고소득자와 저소득자가 동일한 대출 한도를 적용받는 불합리성
- 실제 상환능력과 대출 규모 간 불일치 문제

**DTI 도입 효과:**
- **소득 기반 대출 한도** 설정으로 상환능력 중심 평가
- **과도한 가계부채 증가 억제**
- **주택자금 수요 적정화**

## 3. LTV, DTI, DSR 비교

### 3.1 계산 방식의 구체적 차이

| 대출 유형 | LTV 반영 | DTI 반영 | DSR 반영 |
|-----------|----------|----------|----------|
| **주택담보대출** | 대출원금 전액 | 원리금 상환액 | 원리금 상환액 |
| **신용대출** | 미반영 | 이자 상환액만 | 원리금 상환액 |
| **카드론** | 미반영 | 이자 상환액만 | 원리금 상환액 |
| **자동차할부** | 미반영 | 이자 상환액만 | 원리금 상환액 |

### 3.2 실제 계산 비교 (동일 조건)

**차주 정보:**
- 연소득: 6,000만원
- 주택 감정가: 8억원
- 신청 주택담보대출: 5억원 (금리 4%, 30년)
- 기존 신용대출: 3,000만원 (금리 8%, 5년)

**계산 결과:**

```
LTV = 5억원 ÷ 8억원 × 100 = 62.5% ✅

DTI = (주택대출 원리금 3,400만원 + 신용대출 이자 240만원) ÷ 6,000만원 × 100 = 60.7%

DSR = (주택대출 원리금 3,400만원 + 신용대출 원리금 740만원) ÷ 6,000만원 × 100 = 69.0%
```

**결과 분석:**
- **LTV**: 통과 (62.5% < 70%)
- **DTI**: 통과 (60.7% < 70%)  
- **DSR**: 한도 초과 (69.0% > 60%) → **대출 승인 불가**

:::warning[가장 엄격한 기준]
세 지표 중 **하나라도 한도를 초과하면 대출이 불가**합니다. 일반적으로 DSR이 가장 엄격한 기준이 됩니다.
:::

## 4. DTI 규제 현황

### 4.1 현행 DTI 한도

| 지역 구분 | DTI 한도 | 비고 |
|-----------|----------|------|
| **투기지역** | 60% | 서울 강남 3구 등 |
| **투기과열지구** | 60% | 규제지역 대부분 |
| **조정대상지역** | 70% | 수도권 일반 지역 |
| **일반 지역** | 70% | 비수도권 |

### 4.2 우대 조건

**생애최초 주택구입:**
- DTI 한도 **10%포인트 완화**
- 예: 일반 60% → 생애최초 70%

**다자녀 가구:**
- 추가 **5%포인트 완화**
- 3자녀 이상 가구 대상

## 5. 주택 구매 전략

### 5.1 DTI 기반 구매 가능 금액 계산

**Step 1: 가용 상환액 계산**
```
연소득 × DTI 한도 = 최대 연간 상환 가능액
예: 8,000만원 × 60% = 4,800만원
```

**Step 2: 기존 대출 이자 차감**
```
4,800만원 - 기존 대출 이자 = 주택대출 가용 상환액
예: 4,800만원 - 500만원 = 4,300만원
```

**Step 3: 대출 가능 금액 산정**
```
금리 4%, 30년 기준: 약 8억원 대출 가능
→ LTV 70% 적용 시 구매 가능 주택가격: 약 11억 4천만원
```

### 5.2 DTI 최적화 전략

**1. 구매 전 준비**
- 기존 고금리 대출 미리 정리
- DTI 우대 조건 충족 여부 확인
- 소득 증빙 자료 완비

**2. 대출 구조 설계**
- 혼합금리(고정+변동) 활용
- 거치기간 적절히 활용
- 중도상환 계획 수립

**3. 장기 관리 방안**
- 소득 증가에 따른 중도상환
- 금리 하락 시 대환대출 검토
- DTI 여유분 활용한 추가 투자

:::tip[DTI 활용 극대화]
DTI는 **주택 구매력을 결정하는 핵심 지표**입니다. 구매 전 DTI를 최적화하면 더 큰 규모의 주택 구매가 가능합니다.
:::
