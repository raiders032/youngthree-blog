## 1. 기준가격이란?

- 펀드 기준가는 펀드를 사고팔 때 기준이 되는 가격입니다.
  - 주식의 주가처럼 펀드의 현재 가치를 표시하는 지표입니다.
- 1좌의 단위는 너무 작아 불편하므로 기준가를 계산할 때는 기준가격에서 일반적인 기준가산정단위 ‘1,000’을 곱한 1000좌 단위로 표기합니다. 
- 표기할 때는 1,000좌 단위로 원미만 셋째자리에서 4사 5입하여 원미만 둘째자리까지 표기합니다.

:::info[기준가격 표시 방식]

일반적으로 "1,000좌당 얼마"로 표시되며, 최초 설정 시 1,000원에서 시작하는 경우가 많습니다.

:::

## 2. 기준가격 계산 공식

- 기준가격은 다음과 같이 계산됩니다
- 먼저 순자산 총액을 총좌수로 나눠 1좌당 가치를 계산합니다.
- 그 다음 1좌당 가치를 1,000으로 곱해 기준가격을 계산합니다.

```
기준가격 = (순자산 총액 ÷ 총좌수) × 1,000
```

### 2.1 계산 예시

- **순자산 총액**: 10억 원
- **총좌수**: 1,000만 좌

```
기준가격 = (1,000,000,000원 ÷ 10,000,000좌) × 1,000
        = 100원 × 1,000
        = 100,000원
```

이 경우 기준가격은 **100,000원**입니다.

:::note[순자산 총액이란?]

순자산 총액 = 펀드가 보유한 모든 자산(주식, 채권 등)의 시가 총액 - 부채 - 운용보수 등

매일 장 마감 후 계산되어 다음 영업일에 공시됩니다.

:::

## 3. 기준가격으로 좌수 계산

- 투자금액에 따라 받게 되는 좌수는 다음과 같이 계산됩니다.

```
매입 좌수 = (투자금액 ÷ 기준가격) × 1,000
```

### 예시 1: 기준가격이 높을 때

- **투자금액**: 100만 원
- **기준가격**: 1,350원

```
매입 좌수 = (1,000,000원 ÷ 1,350원) × 1,000
         = 740.74 × 1,000
         = 740,740좌
```

### 예시 2: 기준가격이 낮을 때

- **투자금액**: 100만 원
- **기준가격**: 950원

```
매입 좌수 = (1,000,000원 ÷ 950원) × 1,000
         = 1,052.63 × 1,000
         = 1,052,630좌
```

## 4. 평가금액 계산

- 보유 중인 펀드의 현재 가치는 다음과 같이 계산됩니다.

```
평가금액 = (총보유좌수 × 현재 기준가격) ÷ 1,000
```

### 4.1 실전 예시

- **매입 시**
  - 투자금액: 500만 원
  - 매입 기준가격: 1,200원
  - 매입 좌수: (5,000,000 ÷ 1,200) × 1,000 = **4,166,666좌**
- **평가 시**
  - 현재 기준가격: 1,500원
  - 평가금액: (4,166,666 × 1,500) ÷ 1,000 = **6,249,999원**
- **수익률**: (6,249,999 - 5,000,000) ÷ 5,000,000 × 100 = **25%**

:::warning[기준가격 반영 시점]

기준가격은 매일 장 마감 후 계산되어 다음 영업일에 공시됩니다. 따라서 오늘 매수/매도 신청 시, 오늘 장 마감 후 계산된 기준가격이 적용됩니다.

:::

## 5. 매입 시 기준가격 적용일

펀드 매입 시 적용되는 기준가격은 **펀드 종류**와 **매입 청구 시점**에 따라 달라집니다.

### 5.1 펀드 종류별 기준가격 적용 기준

| 구분 | 시간 | T(당일) | T+1일(2영업차) | T+2일(3영업차) |
|------|------|---------|----------------|----------------|
| **주식 50% 이상** | 오후 3시 30분 이전 | 매입청구일 | 적용기준가/<br/>펀드매입일 | - |
| | 오후 3시 30분 경과 후 | 매입청구일 | - | 적용기준가/<br/>펀드매입일 |
| **주식 50% 미만,<br/>채권형** | 오후 5시 이전 | 매입청구일 | 적용기준가/<br/>펀드매입일 | - |
| | 오후 5시 경과 후 | 매입청구일 | - | 적용기준가/<br/>펀드매입일 |
| **MMF** | 오후 5시 이전 | 매입청구일 | 적용기준가/<br/>펀드매입일 | - |
| | 오후 5시 경과 후 | 매입청구일 | - | 적용기준가/<br/>펀드매입일 |

:::info[표 읽는 방법]

- **T(당일)**: 매입 청구를 한 당일
- **T+1일**: 매입 청구일로부터 1영업일 후
- **T+2일**: 매입 청구일로부터 2영업일 후
- **적용기준가**: 실제로 적용되는 기준가격이 확정되는 날
- **펀드매입일**: 해당 좌수가 계좌에 반영되는 날

:::

### 5.2 주의사항

- 펀드 종류가 같아도 **개별 펀드마다 기준이 다를 수 있습니다**.
- 매입 전 반드시 해당 펀드의 **투자설명서**에서 기준가격 적용일을 확인하세요.
- 공휴일이나 휴장일은 영업일에서 제외되므로 적용일이 더 늦어질 수 있습니다.
- 일부 해외펀드의 경우 오후 5시 이전에 매입 청구를 하면 제3영업일(T+2일) 기준가격이 적용되며, 오후 5시 이후에 매입 청구를 하면 제 4영업일(T+3) 기준가격이 적용되는 등개별 펀드마다 기준이 다를 수 있습니다.
